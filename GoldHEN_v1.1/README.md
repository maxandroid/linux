## Features
- Homebrew Enabler
- Debug Settings
- VR Support
- Remote Package Install
- Rest Mode Support
- External HDD Support
- Official External HDD Format Support
- Debug Trophies Support
- sys_dynlib_dlsym Patch
- UART Enabler
- Never Disable Screenshot
- Remote Play Enabler
- Persistent FTP on 2121 port (Rest Mode Support)
- FW Update Block
